# LibreNotes

**LibreNotes** &ndash; мобильное приложение для хранения заметок с возможностью их синхронизации между устройствами.

## Технологии

Используется фреймворк для разработки кроссплатформенных мобильных приложений [Flutter](https://flutter.dev/) (язык программирования [Dart](https://dart.dev/)). Синхронизация данных приложения с сервером происходит с использованием [REST-like](https://restfulapi.net/) API.
